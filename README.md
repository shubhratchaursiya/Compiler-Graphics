## üíª Programs Included

Each program demonstrates a fundamental concept of Computer Graphics:

1. **DDA Line Drawing Algorithm**  
   Draws a straight line using incremental calculations.

2. **Bresenham‚Äôs Line Drawing Algorithm**  
   Efficient integer-based line drawing algorithm.

3. **Midpoint Circle Drawing Algorithm**  
   Draws a circle using midpoint decision parameters.

4. **Midpoint Ellipse Drawing Algorithm**  
   Generates an ellipse using region-based logic.

5. **Cohen‚ÄìSutherland Line Clipping Algorithm**  
   Clips lines outside a rectangular window.

6. **Liang‚ÄìBarsky Line Clipping Algorithm**  
   Efficient parametric line clipping technique.

7. **Flood Fill Algorithm**  
   Fills a region starting from a seed point.

8. **Boundary Fill Algorithm**  
   Fills a region until a boundary color is reached.

9. **2D Transformations**  
   Implements translation, rotation, and scaling.

10. **Bezier Curve Generation**  
    Draws curves using control points.

11. **B-Spline / Spline Curve Program**  
    Generates smooth curves for modeling.

12. **Window-to-Viewport Transformation**  
    Maps world coordinates to screen coordinates.

13. **Basic Animation Program**  
    Demonstrates simple object movement.
